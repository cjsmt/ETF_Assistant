主观+量化ETF研究框架整理

一、摘要与核心结论
1.1 研究动机、策略定位（为什么做）
本策略旨在构建一套“趋势-资金共识”双核驱动的行业轮动体系，核心目标是解决传统基本面策略的“数据滞后性”与“情绪捕捉盲区”。
1.1.1 传统策略的滞后性痛点
传统投资主要依赖“后视镜”数据，难以应对快速轮动的市场：
基本面数据滞后：净利润、ROE等核心指标受限于季报披露周期（滞后1-4个月）。当亮眼财报公布时，预期往往早已在股价中兑现，甚至出现“利好兑现即利空”。
估值数据静态：PE/PB依赖于历史财务数据，难以反映市场对未来成长性的动态重估。
1.1.2 量价因子解决滞后性问题的机制
我们引入“趋势（价）+ 筹码（量）”的双核因子，利用市场交易行为的实时性与博弈特征，提前捕捉主力资金动向。
价格因子（趋势强度），预期的先行指标
核心逻辑：价格是所有投资者对未来预期的实时总和，领先于财报信号。
因子升级：我们不只看单纯涨跌，而是引入均线排列形态（MA Score）与12个月动量。
作用：筛选出当前市场资金“用脚投票”选出的最强主线。均线多头排列代表全周期持仓获利，抛压最小，趋势惯性最强，从而在基本面数据落地前提前锁定领涨板块。
资金因子（筹码结构）：情绪的博弈监控
核心逻辑：单纯的“成交量放大”无法区分是主力建仓还是散户接盘。我们引入“资金性质”的维度进行博弈分析。
因子升级：重点采用ETF资金流逆向指标与北向/主力资金流。
ETF资金流（逆向）：研报数据显示，散户在ETF交易中呈现显著的“逆势操作”特征（越跌越买，越涨越卖）。因此，“价格上涨 + ETF资金流出”往往代表散户恐高离场、筹码向机构集中的健康状态；反之则需警惕。
波动率收敛：关注资金流波动率的下降，这代表多空分歧减少，资金共识形成。
作用：剔除“虚假繁荣”，规避散户扎堆的高风险区域，解决市场情绪研判的滞后问题。
双核联动（择时）：趋势与共识的交叉验证
核心逻辑：结合量价实现“去伪存真”。
判定机制：
真突破（买入）：趋势向上（均线多头） + 筹码结构优化（主力流入/散户流出）。这是有资金实打实支持的强势信号。
假诱多（规避）：趋势向上（短期暴涨） + 筹码结构恶化（ETF巨量流入/散户狂热）。这预示着筹码松动，潜在抛压巨大。
通过这种严格的纪律性筛选，策略能够在行情启动初期捕捉信号，并在情绪过热前及时撤退，彻底解决等待信息确认后的时机滞后问题。
1.1.3 策略目标：主观与量化的完美分工
景气度/宏观（主观）：负责判断“谁未来会赚得多”，确定行业方向的确定性（定性分析）.
量价因子（量化）：负责判断“什么时候买入最合适”，实现交易的实时性（定量分析），解决时机滞后.
通过这种结合，策略能够将基本面的长期逻辑（主观定调）与交易面的实时信号（量化择时）相结合，从而获得在行业轮动中的 Beta 收益。

1.2 核心优势
本策略的核心优势在于完美结合了 主动投资的智慧 与 量化模型的效率和实时性，有效克服了传统策略的滞后性痛点 。
具体来说，策略通过量价因子（趋势与拥挤度）捕捉市场参与者的实时交易行为和资金博弈情况 ，解决了基于滞后财报信息做决策的难题 。这使得投资决策既有基本面和宏观逻辑的支撑（由主观判断和景气度因子提供），又有交易层面的实时敏锐度（由量价因子提供），从而在高效率地实现行业轮动中捕捉 Beta 收益 。

二、研究背景与策略设计
2.1 主观定调（基于何种背景，可选美联储降息收官背景）
我们基于“内外双循环”的宏观视角，预先圈定本期的“核心观察池”，为后续量化择优指明方向。
2.1.1 外需视角：紧盯美债利率下行（锁定“出口链”池）
宏观背景：参考申万宏源宏观策略，美联储降息周期下，10年期美债利率趋势下行，将依次激活美国的“零售销售 -> 地产销售 -> 制造业投资”需求。
定池逻辑：
地产后周期池：美国地产销售复苏具有确定性传导，重点圈定家电（出海龙头）、家居用品、建材行业。
再工业化池：美国制造业回流带来设备投资需求，重点圈定工程机械、电力设备行业。
资源品池：美元信用重构与工业需求复苏，重点圈定有色金属（铜/铝）行业。
2.1.2 内需视角：紧跟财政发力方向（锁定“政策链”池）
宏观背景：2025年国内政策主线明确为“财政发力”与“安全发展”并重。
定池逻辑：
存量挖掘池（以旧换新）：受益于国债资金支持的“两新”政策，重点圈定汽车、家电、机械设备。
安全博弈池（自主可控）：地缘政治背景下的刚性投入领域，属于独立景气赛道，重点圈定半导体、军工、信创（计算机）。
制度红利池（防御底仓）：若市场处于震荡弱势，基于并购重组预期与高股息逻辑，圈定公用事业、银行、央企基建作为防御性备选池。

2.2 介绍模型、因子
量化模型：因子构建与落地打法
我们构建的“量价双核”模型，核心在于解决传统基本面数据的滞后性。我们利用“价”确认趋势的强度，利用“量”确认资金的共识与性质。

2.2.1 具体使用了哪些因子 (Factor Construction)
基于研报的深度挖掘，我初选出两组最有效的因子：

A. 趋势因子（价 - Price）：寻找“强趋势”
因子1：均线排列形态得分 (MA Score)
来源： 《趋势明确与资金共识...华西证券》 
定义： 考察短期(10日)、中期(20日)、长期(60日)均线的排列。多头排列（短>中>长）计正分，空头排列计负分。
逻辑： 均线不仅过滤了噪音，还代表了持仓成本的分布。多头排列意味着所有周期的持仓者都盈利，抛压最小，趋势最强 。
因子2：12个月动量 (12M Momentum)
来源： 《六维因子...华鑫证券》 
定义： 过去12个月的累计涨跌幅（可剔除最近1个月以去除反转效应）。
逻辑： 长期动量是A股最显著的异象之一，强者恒强在行业层面尤为明显 。

B. 拥挤度/资金因子（量 - Volume）：寻找“真共识”
因子3：ETF资金流逆向指标 (Retail Sentiment)
来源： 《ETF资金流的逆势特性...浙商证券》 
定义： 个人投资者占比较高的ETF资金净流入。
逻辑： 这是一个反向指标。研报证明，散户在ETF上呈现显著的“逆势交易”特征（越跌越买，越涨越卖）。如果某行业价格上涨但ETF资金大幅流出，往往是主力拉升、散户离场的健康上涨信号；反之，如果价格下跌且ETF大幅流入，可能是散户在“接飞刀” 。
因子4：北向资金/主力资金净流入 (Smart Money)
来源： 《六维因子...华鑫证券》 
定义： 北向资金或大单/超大单的净流入额。
逻辑： 代表“聪明钱”的流向，与散户资金形成对冲，用于确认趋势的真实性 。

2.3 策略执行规则：因子的联动逻辑 (Linkage Logic)
单纯追涨易导致高位被套，单纯看资金易陷入“抄底陷阱”。本策略将价格形态与资金特征深度融合，构建了一套“趋势+共识”的四象限筛选体系。
我们的核心逻辑是寻找“趋势向上”且“资金产生共识（即主力资金进场、散户资金离场或分歧收敛）”的优质标的。具体判定标准如下：

黄金配置区（重仓买入）：趋势确立，筹码稳固 当板块处于均线多头排列且动量强劲的状态，同时伴随主力资金流入、ETF资金流出（反映散户“恐高”离场）时，意味着筹码已完成向机构端的充分交换，上涨趋势真实有效且具备持续性，发出重仓买入信号。
左侧观察区（重点关注）：分歧收敛，变盘在即 当板块均线纠缠且动量较弱，但资金流波动率显著下降时，表明市场多空分歧正在减小，资金共识逐渐形成。这往往是行情启动的前夕，建议左侧关注，等待突破确认。
高危警示区（分批止盈）：情绪过热，筹码松动 若板块价格短期暴涨且动量极强，同时出现ETF巨量流入（反映散户狂热追涨），表明市场情绪已达非理性高点，潜在抛压巨大，建议分批止盈以规避见顶风险。
垃圾规避区（坚决回避）：下跌中继，接盘风险 当板块处于均线空头排列且动量走弱，同时出现主力流出、ETF大幅流入（反映散户逆势“抄底”）时，通常为下跌中继形态，属于典型的“接盘侠”陷阱，建议坚决回避。



2.4 量价双核”模型回测表现

三、 体系升维：主观赋能与最终落地 (Quantamental Execution)
本章阐述如何构建“量化筛选广度 + 主观把控深度”的漏斗式决策体系，将量化信号转化为可执行的ETF投资组合。
3.1 流程架构：漏斗式筛选机制 (The Funnel Process)
我们将投资决策过程标准化为三层过滤漏斗，逐级收敛标的范围：
第一层（全市场扫描）： 利用量价双核模型，对全市场一级行业指数进行纪律性打分，输出“高胜率观察池”（约3-5个行业）。
第二层（主观风控）： 引入宏观与基本面逻辑，对观察池进行“宏观定调”与“排雷否决”，剔除逻辑相悖的标的。
第三层（产品落地）： 在确定的行业赛道中，根据流动性与规模规则，锁定最优的交易工具（ETF）。

3.2 主观校验：宏观排雷 (The Subjective Filter)
量化模型擅长捕捉趋势，主观研究擅长识别拐点与黑天鹅。本环节核心在于“纠偏”。
3.2.1负面清单排雷 (Veto Power) 执行“一票否决制”，剔除存在硬伤的量化高分标的：
关税与地缘风险：对于高度依赖美国市场且无可替代性弱的纯消费出口行业（如部分纺服、玩具），若面临关税政策恶化，即使量价齐升也坚决回避 。
政策风险：行业面临强监管或政策打压（如教培、反垄断），即使量价齐升也坚决回避。
逻辑证伪：量化捕捉到的上涨若仅为“超跌反弹”且无基本面支撑（如产能过剩未出清的光伏），应视为“假突破”并剔除。

3.3 ETF精准映射：产品优选规则 (ETF Mapping)
行业指数不等于可交易资产。基于华鑫证券ETF轮动策略 ，执行以下映射标准：
3.3.1 流动性与规模双门槛
流动性过滤：剔除日均成交额 < 5000万 的ETF，确保大资金进出无滑点冲击。
规模安全线：优选规模 > 2亿 的非迷你基金，规避清盘风险。在同类ETF中，优先选择规模最大、跟踪误差最小的“龙头ETF”。

3.3.2 映射替代策略
精准映射：若行业有对应纯粹ETF（如“证券”→“证券ETF”），直接配置。
模糊/增强映射：若细分赛道无纯粹ETF，或追求超额收益，可采用Smart Beta ETF替代。
（示例：若看好“高端制造”但无直接ETF，可配置“广发成长智选” 或“广发制造智选”  等增强型产品作为替代工具。）
